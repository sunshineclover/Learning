# 具体步骤

1. 冬瓜切好片，备用
2. 热锅后加油，之后下入冬瓜片，翻炒两分钟，之后可以加一点点水。
3. 加完水后用盖子焖1分钟，之后再上下翻一次后，焖1分钟。
4. 放入蚝油，一点生抽，盐（少量），味精，鸡精后，拌匀，即可出锅装盘。





# 注意事项

1. 第二步时为了防止冬瓜炒糊，**可以加一点点水**。

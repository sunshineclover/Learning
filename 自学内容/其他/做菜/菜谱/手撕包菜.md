# 具体步骤

1. 去掉最上两层后，剩下部分手撕下来。最好只留叶子，不要菜梆子
2. 撕好后清洗滤水，准备大蒜，干辣椒，花椒（可以不放）。
3. 锅烧热后放油，立刻放入花椒干辣椒，等到辣椒微微发黑红放入蒜，再立马放入包菜。
4. 菜炒蔫了后放盐，味精/鸡精，即可出锅。




# 注意事项

1. 去掉一两层的目的：质地较老，口感较差。
2. 菜梆子不易炒熟入味，可以丢掉。

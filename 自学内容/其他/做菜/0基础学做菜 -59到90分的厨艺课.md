# 课程笔记

## 一.  初入厨房

本部分主要介绍基础的炒菜方法及常用调配料。主要是对厨房内各种物品的科普及简单介绍。

### 1. 认识基本的厨具    ——视频ep.2 

1. 锅
- 炒锅：更推荐**铁锅**（导热性能好），而不是不粘锅。
- 汤锅：煮面条/烧水
- 平底不粘锅：煎饼/煎肉
- 小奶锅：做快手菜

2. 锅盖
- 长时间加热使用：炖、焖、蒸、煮

3. 锅铲
- 铁锅可以用铁锅铲
- 不粘锅一定要用木铲/硅胶锅铲，防止把涂层刮掉。

4. 汤勺
- 用来盛汤

5. 菜刀
- 切刀：刃薄锋利，用来切菜。
- 砍刀：刃厚较钝，用来砍有骨头的原料（鸡鸭等）

6. 菜板
- 使用木制菜板，用完晾干防止发霉
- 不能选塑料菜板，容易滋生细菌。

7. 去皮器，滤网，毛巾，钢丝球
- 去皮器：给土豆之类的削皮
- 滤网：洗菜用
- 毛巾：新毛巾洗碗，旧毛巾擦灶台，钢丝球（不粘锅，电饭锅，有涂层的锅不能用）

8. 蒸架，漏网

### 2. 家中必备的调料   ——视频ep.3 

**生抽，陈醋，油，大蒜，生姜，淀粉，鸡精，味精，盐**，料酒，豆瓣酱，干辣椒，白糖，胡椒粉等

### 3.认识盐  ——视频ep.4-5

放盐的原则：**宁少勿多**。
注意：**倒油之前锅里一定不能有水！**

例： 尖椒蛋饼，干煸土豆丝，**[[煎鸡蛋]]**。
例：[[荷包蛋]]

### 4. 味精与鸡精 ——视频ep.6-8

例：[[炒空心菜]]。
注意：炒菜的时候，建议鸡精与味精同时使用，且用量大约为盐用量的1/5-1/4。

**炒菜公式1：油+菜+盐+鸡精/味精->出锅**
应用实例：[[炒菠菜]]
调料用量：味精是盐的1/3-1/4，鸡精是味精的1/3-1/4

### 5.  干辣椒、大蒜与花椒 ——视频ep.9

- 川菜三剑客:花椒、干辣椒、蒜
- 一般来说**素菜放蒜**，**肉菜放姜，蒜**，**鱼放葱，姜，蒜**(清汤鱼 只要葱和姜)
- 火力：
1. 煎东西就尽可能中小火，避免食材外面都快糊了芯子还没熟
2. 再就是正式开炒前煸料(如葱姜蒜花椒干辣椒大料豆瓣酱大酱黄豆酱等)，肯定也要用中小火，避免调料炸糊了。
3. 中大火那就是你炒青菜时、往锅里烹料酒、酱油、香醋等酱料时，再就是炖煮过后，快速收汁。

例：[[炒菜苔]]，[[手撕包菜]]（莲花白就是包菜），与之类似的还有炒豆腐菜（落葵）->(洗菜需要去掉老茎）和炒小白菜。


## 二. 刀工

### 1.  如何选刀 ——视频ep.11

片刀（锋利，刃薄的中式刀）和砍刀。 菜板选择木头菜板（最好一大一小）

### 2.  切菜姿势与火候的选择 ——视频ep.12-ep.13

1. 站立姿势：双脚站开与肩同宽即可。
2. 握刀姿势：食指按在刀背/拇指按在刀背/只握刀把，**皆为错误姿势**。
**正确的姿势**：拇指扶着，背面食指可以微微弯曲。（类似乒乓球横板）
3. 切菜姿势：左手**中指第一个关节支撑**刀，注意**手指要往里握（千万不要平着切）**，**手掌需要靠住食材/菜板（不要悬空）**，且刀抬起的高度**不要超过中指的第一个关节**。
4. 注意：切菜左手的作用：支撑/量距离（食材要切多大，就留出多大的距离）。

火候的选择：
四个挡位：小火，中小火，中火，大火。
- 小火：炖汤，放调料煎荷包蛋
- 中小火：炖煮蒸菜
- 中火：炒菜
- 大火：炒菜/把汤烧开


常见例子：
- 炖鸡：大火烧开后小火慢炖
- 红烧肉：大火烧开后小火慢炖再大火收汁

例：[[南瓜汤与冬瓜汤]]


### 3.  切片 ep.14
- 中指抵住刀，直接开切，注意薄厚均匀。
- 切三退一/切五退一（声音好听）

耗油（细腻的咸鲜感），生抽（咸鲜）
[炒冬瓜](菜谱/炒冬瓜.md)

### 4. 切丝 ep.15

- 先切片（**让片尽量均匀，否则丝也不均匀，具体看注意事项**），再切丝
- 一边宽，一边窄的情况：先再宽的那边切两三刀，再将其作为底部（让土豆稳定，固定住），就可以切片了。
- 片切好后，把片铺起来，直接切丝即可
- 克服切片不均匀的方法：**刀锋利；握刀要稳**。
- 最后小的那几片的切法：把指甲抠进去固定。

[青椒炒土豆丝](菜谱/青椒炒土豆丝.md)

### 5. 切丁 ep.16

- 切丁之前，先将食材切成片，再切成丝后，最后放整齐即可切成丁。

[扬州炒饭](菜谱/扬州炒饭.md)


### 6. 养锅技巧 ep.17

- 炒菜的锅，只用来炒菜，不要用来烧水和煲汤。
- 避免锅里长时间有水，可以倒扣或放入凉台将水控干。

[葱油饼](菜谱/葱油饼.md)


### 7. 切末 ep.18

- 可以直接剁（到处都是）
- 可以切小一点在剁
- 左右摇摆来回切
- 竖着切完横着切

### 8. 滚刀块 ep.19

- 先斜着切一刀，再向内滚90度后斜着切一刀，保持一致即可切完。

[素炒胡萝卜](菜谱/素炒胡萝卜.md)

### 9.  常用调料的购买与大致使用方法 ep.20-ep.21

- 油：换着花样吃最好。
- 盐：加碘食用盐，腌制盐是做咸菜泡菜的，粗盐不适合炒菜。
- 味精：买标注99\%的味精（谷氨酸钠占比99）。小颗粒状比大颗粒状方便一些。适合买袋装的（便宜）
- 鸡精：莎麦/太太乐。
- 酱油：海天金标生抽，东古生抽等，老抽上色。
	- 部分人的配法：一瓶特级酱油（有色有味）一瓶老抽（有色）一瓶生抽（有味）一瓶蒸鱼豉油，就够了
	- 部分人的配法：一瓶冰糖老抽、一瓶东古一品鲜、一瓶味极鲜、一瓶蒸鱼豉油、一瓶淡盐生抽。
- 醋：陈醋和白醋
	- 陈醋：酸度不同，价格不同。一般家用买一级，二级。
	- 白醋：也称作米醋，度数表明酸度。
- 料酒：用来去腥。不推荐只有原料+焦糖色的料酒。
- 蚝油：买海天的。经常做饭买大瓶，偶尔做饭买小瓶。
- 豆瓣酱：郫县豆瓣酱。（买小瓶的）
	- 产地：四川郫县
	- 颜色：颜色红亮，色泽红润。
- 鲜花椒油/藤椒油
	- 藤椒油清香，麻味不如鲜花椒油。鲜花椒油麻味更足。
- 芝麻油
	- 做火锅蘸料。可以在水煮鱼，麻辣香锅等加入
	- 购买用纯芝麻压榨的最好，
- 干花椒（红花椒与青花椒）
	- 红花椒香，青花椒麻。家用一般选择青花椒。
- 花椒粉（配料只有花椒）
- 白胡椒粉（用来去腥，配料只有胡椒）
- 干辣椒（新一代的干辣椒）
	- 个小的比个大的辣，个细的比个粗的辣。
- 桂皮，八角（大料）：适合炖肉，一般两种混合使用。
- 陈皮：去腥，除膻，增香，解腻，开胃。煮牛羊肉可以放一块。（可以吃橘子后皮留下晾干）
- 淀粉：让肉质更鲜嫩。常见的有红薯淀粉，玉米淀粉。
- 泡椒（买点散装即可，或买泡野山椒，可以放点泡椒水）
- 白糖：买点散装即可
- 白芝麻：去皮白芝麻与没去皮白芝麻
	- 前者更佳，看起来更为美观
	- 生的白芝麻不香，需要炒熟后的，呈米黄色的那种。
	- 一般来说，凉菜用熟芝麻，需要炒的菜用生芝麻，黑芝麻一般不用。
- 十三香：认准王守义十三香。（麻辣干香的菜）
- 孜然粉：烧烤。
- 生姜：去腥增香。个头小的更香，但个头大的洗干净的更方便。
	- 一般用的是老姜，仔姜的去腥增香一般，只在特定的菜品使用。
- 大蒜：去腥增香。买最普通的就可以，需要放冰箱保存防止发芽。
- 小葱：去腥增香。
- 大葱：去腥增香（去腥非常好）。
- 香菜。
- 芹菜：适合鱼或者泡椒配合使用，一般适合买小白芹。
- 蒜苗：有独特的香气，适合回锅肉。腊肉，牛肉，猪肝，鸡蛋等放一点。

# 课程评价

课程适合接在翁恺后看，其实对于课堂学习已经没什么帮助了，但对于C++的了解还不错。

主要补充了C++独有的部分，如模板，面向对象及部分STL知识，可以作为一个学习的参考。

整体讲的还不错，把重要的内容讲的通俗易懂，如虚函数，运算符重载等。

推荐翁恺C语言后看，对于后续Java的学习也有很大作用（毕竟OOP思想一致）

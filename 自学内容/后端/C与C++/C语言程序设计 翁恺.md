# 课程评价

没什么可说的，第一次接触C语言的话，最推荐的网课。

除此之外，C语言还推荐阅读C Primer Plus，但相比该网课难度较大，且部分不易理解。

仅为入门编程大门的话，强烈推荐该网课。

若为入门计算机的大门，更推荐从计算机相关导论课开始（如CS50系列）
